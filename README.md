# Hardware-Accelator
This Hardware Neural Network engine is a sophisticated Verilog implementation designed to execute a multi-layer feed-forward inference process through a dedicated finite state machine. The system integrates several specialized hardware components, including a weight/bias memory for parameter storage, a multi-port register file for data handling, and dual Arithmetic Logic Units alongside Multiply-Accumulate units to accelerate core computations. The architecture is governed by a central controller that manages the data flow across distinct stages, beginning with a loading phase where weights are fetched from ROM and ending with a post-processing stage that produces the final 32-bit signed output. A robust error-tracking system is embedded within the logic to monitor every stage of the process for arithmetic overflows and zero-results, specifically flagging which part of the pipeline-such as the input or output layer—encountered a mathematical exception. The design’s reliability is verified through an automated testbench that compares the hardware's real-time performance against a reference model. This verification suite subjects the engine to various test cases, including random values and extreme boundary conditions, to ensure that the overflow logic and final outputs remain consistent under computational stress. Additionally, the hardware architecture utilizes a modular approach where the mac_unit combines multiplication and addition into a single optimized block, facilitating the efficient dot-product calculations necessary for neural networks.
